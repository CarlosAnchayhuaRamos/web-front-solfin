# SOLFIN API

Backend base en NestJS con Prisma y PostgreSQL.

## Comandos

```bash
npm install
npm run prisma:generate
npm run prisma:push
npm run start:dev
```

## Estructura inicial

```text
server/
  prisma/schema.prisma
  src/
    app.controller.ts
    app.module.ts
    app.service.ts
    main.ts
    prisma/prisma.service.ts
```
